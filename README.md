# phaser-books-de
Deutsche Bücher über das Spiele programmieren mit Phaser

[Phaser](http://phaser.io/) ist ein großartiges, offenes HTML5 Spiele-Framework für Canvas und WebGL welches im Browser und auf mobilen Geräten läuft. Es verfügt über sehr viele Beispiele, leider alle nur auf Englisch. An diesem Umstand möchte ich etwas ändern und die Einstiegshürde mit deutschen Ressourcen senken.
